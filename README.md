# Email Spam Classifier

A machine learning project to classify SMS/email messages as spam or ham using Python, TF-IDF, and Multinomial Naive Bayes (scikit-learn).

## Features
- Preprocessing and vectorization with TF-IDF
- Model training using Naive Bayes
- CLI-based predictor for real-time message classification

## Tech Stack
Python, Pandas, scikit-learn, joblib

## Accuracy
~98% on SMS Spam Collection dataset
